
# RetroLemonPlayer 🍋

RetroLemonPlayer is a nostalgic media player with a Windows 7 vibe, created to play:

- MP3 audio files  
- MP4 video files  
- Images (JPG/PNG)

It includes instant timeline seeking, Windows 7 hover glow, Aero blur effect, and smooth playback similar to MPC-HC.

---

## Features

- 🎵 Audio & video playback  
- 🖼 Image viewing  
- ⏱ Instant timeline seek  
- 💨 Smooth playback like MPC-HC  
- ✨ Windows 7 hover glow & Aero blur  
- 🔊 Volume control  
- 🖱 Drag window anywhere  
- 🔳 Minimize / Maximize / Close buttons like Windows 7  

---

## Installation

Download the latest **RetroLemonPlayer_Setup.exe** from github. Click the green buton named 'Code' then click download as zip. Extract and run net6.0-windows.exe . Follow the setup wizard to install RetroLemonPlayer.
